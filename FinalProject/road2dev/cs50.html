<What is CS50>

It's Harvard University's introduction to the intellectual enterprises of computer science and the art of programming for majors and non-majors alike, with or without prior programming experience. An entry-level course taught by David J. Malan, CS50x teaches students how to think algorithmically and solve problems efficiently. Topics include abstraction, algorithms, data structures, encapsulation, resource management, security, software engineering, and web development. Languages include C, Python, SQL, and JavaScript plus CSS and HTML. Problem sets inspired by real-world domains of biology, cryptography, finance, forensics, and gaming. The on-campus version of CS50x , CS50, is Harvard's largest course - CS50. The course is divided in 11 weeks, is self-paced with a time commitment between 10 and 20 hours per week if you choose only one path to complete it. The paths are defined by "less confortable" and "more confortable", each one with a series of problems divided in three sections: "Practice Problems", "Lab" and "Problem Set of The Week". I have choosed to take both paths, this way the duration for me was almost the double.

Week 0 - Scratch

Scratch is the world's largest coding community and a programming language with a simple visual interface that lets young people create digital stories, games, and animations. Scratch is designed, developed, and moderated by the Scratch Foundation, a non-profit organization. Scratch promotes computational thinking and problem-solving skills; creative teaching and learning; self-expression and collaboration; and equity in computing. In this week, although was required a simple project with Scratch, i decided to invite my brother and Visual Designer to make a more elaborated project, this way was born "Fulano's Game" and you can play it clicking here: Fulano's Game.
The main subjects of this week are: Computer Science. Computational Thinking. Problem Solving: Inputs, Outputs. Representation: Unary, Binary, Decimal, ASCII, Unicode, RGB. Abstraction. Algorithms. Running Times. Pseudocode. Scratch: Functions, Arguments, Return Values; Variables; Boolean Expressions, Conditionals; Loops; Events; Threads.

Week 1 - C

C is a general purpose, structured, imperative, procedural compiled programming language, standardized by the International Organization for Standardization, created in 1972 by Dennis Ritchie at AT&T Bell Labs for the development of the Unix operating system - Wikipedia. Whay does they teach us C Well, it's one of the most popular programming language, by learning C, you will have an easier time learning other programming languages like C++, C#, Java, Python, and so on, thanks to the similar syntaxes. The main subjects of this week are: C. Source Code. Machine Code. Compiler. Correctness, Design, Style. Visual Studio Code. Syntax Highlighting. Escape Sequences. Header Files. Libraries. Manual Pages. Types. Conditionals. Variables. Loops. Linux. Graphical User Interface (GUI). Command-Line Interface (CLI). Constants. Comments. Pseudocode. Operators. Integer Overflow. Floating-Point Imprecision. You can see week 1 projects clicking here: Week 1 - C

Week 2 - Arrays

In computer programming, an array is a data structure that stores a collection of elements in such a way that each of the elements can be identified by at least one index or one key. This data structure is also known as indexed variable, vector and matrix - Wikipedia. The main subjects of this week are: Preprocessing. Compiling. Assembling. Linking. Debugging. Arrays. Strings. Command-Line Arguments. Cryptography.

Week 3 - Algorithms

In mathematics and computer science, an algorithm is a finite sequence of executable actions that aim to obtain a solution to a given type of problem. According to Dasgupta, Papadimitriou and Vazirani; "Algorithms are precise, unambiguous, standardized, efficient and correct procedures" - Wikipedia. The main subjects of this week are: Searching: Linear Search, Binary Search. Sorting: Bubble Sort, Selection Sort, Merge Sort. Asymptotic Notation: O, Ω, Θ. Recursion.

Week 4 - Memory

Memory is the electronic holding place for the instructions and data a computer needs to reach quickly. It's where information is stored for immediate use. Memory is one of the basic functions of a computer, because without it, a computer would not be able to function properly - TechTarget. The main subjects of this week are: Pointers. Segmentation Faults. Dynamic Memory Allocation. Stack. Heap. Buffer Overflow. File I/O. Images.

Week 5 - Data Structures

A data structure, in computer science, is a collection of both values and operations. It is a concrete implementation of an abstract data type or a basic or primitive data type - Wikipedia. The main subjects of this week are: Abstract Data Types. Queues, Stacks. Linked Lists. Trees, Binary Search Trees. Hash Tables. Tries.

Week 6 - Python

Python is a high-level, script-interpreted, imperative, object-oriented, functional, dynamically typed, and strong programming language. It was launched by Guido van Rossum in 1991 - Wikipedia. The main subjects of this week are: Python Functions, Arguments, Return Values; Variables; Boolean Expressions, Conditionals; Loops. Modules, Packages.

Week 7 - SQL

Structured Query Language, is a domain-specific language developed for managing relational data in a database management system, or for data stream processing in a data stream management system - Wikipedia. The main subjects of this week are: SQL Tables; Types; Statements; Constraints; Indexes; Keywords, Functions; Transactions. Race Conditionals. SQL Injection Attacks.

Week 8 - HTML, CSS, JavaScript

HTML defines the structure of your content, CSS determines the style and layout, and JavaScript makes the content interactive; therefore, it makes the most sense to learn them in that order. JavaScript incorporates valuable skills such as object-oriented, functional, and imperative styles of programming - CIAT. The main subjects of this week are: Internet Routers; TCP/IP; DNS. HTTP: URLs, GET, POST. HTML: Tags; Attributes. Servers. CSS: Properties; Selectors. Frameworks. JavaScript: Variables; Conditionals; Loops. Events.

Week 9 - Flask

Flask is a small web framework written in Python. It is classified as a microframework because it does not require particular tools or libraries, maintaining a simple but extensible core - Wikipedia. The main subjects of this week are: Flask. Route. Decorators. Requests, Responses. Sessions. Cookies.

Week 10 - Emoji

Emoji is a word derived from the combination of the following terms in Japanese: e + moji. Originating in Japan, emojis are ideograms and smileys used in electronic messages and web pages - Wikipedia The main subjects of this week are: Precision. Unicode: Emoji, Code Points, ZWJ.
